## ğŸš€ Getting Started

### Prerequisites

- **Node.js** (v18 or higher)
- **Git**

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/itzsouravkumar/AdvenOfCode2k25.git
   cd AdventOfCode2025
   ```

2. Navigate to a specific day:
   ```bash
   cd Day-1
   ```

3. Run the solutions:
   ```bash
   node task_1.js
   node task_2.js
   ```
